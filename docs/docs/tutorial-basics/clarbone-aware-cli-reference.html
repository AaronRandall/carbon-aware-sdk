Carbon Aware CLI Reference

The following is the documentation for the Carbon Aware CLI

Formatâ€‹

$ CarbonAwareCLI -t <time> -l <location 1> <location 2> -d <path to data file>

Parametersâ€‹

ShortLongRequired / OptionalDescription
-l--locationRequiredThe location is a comma seperated list of named locations or regions specific to the emissions data provided.
-d--data-fileRequiredPath to the emissions source data file
-t--fromTimeOptionalThe desired date and time to retrieve the emissions for. Defaults to 'now'.
-o--outputOptionalOutput format. Options: console, json. Default is json
-v--verboseOptionalVerbose output
--lowestOptionalOnly return the results with the lowest emissions.

$ ./CarbonAwareCLI -l westus -d "azure-emissions-data.json"
